Features

자동 뉴스 수집

RSS, 웹 크롤링을 통해 AI 관련 최신 뉴스를 매일 자동으로 수집합니다.

LLM 요약 및 분류

GPT-4.1 mini / Gemini Flash를 활용해 뉴스를 요약하고, 카테고리별로 분류 및 중요도를 점수화합니다.

텔레그램 자동 발행

큐레이션된 뉴스를 매일 아침 텔레그램 채널로 자동 발행합니다.

How it works

1

수집 (Crawl)

Python 기반 크롤러가 주요 AI 뉴스 소스에서 최신 기사를 수집합니다.

2

분석 (Analyze)

LLM이 각 기사를 요약하고, 카테고리 분류 및 중요도 점수를 매깁니다.

3

발행 (Publish)

GitHub Actions 스케줄러가 매일 아침 텔레그램 채널로 큐레이션된 뉴스를 발송합니다.
