앤스로픽, Claude Code 데스크탑 앱 및 자동화 루틴 공개

앤스로픽이 완전히 재작성된 Claude Code 데스크탑 앱을 공개했다. 이 앱은 하나의 창에서 멀티 세션을 실행할 수 있으며 통합 터미널, 내장 파일 편집, HTML/PDF 프리뷰, SSH 연결 기능을 제공한다. 또한, 일정이나 API 호출, GitHub 이벤트에 따라 자동 실행되는 클라우드 기반 코드 자동화 기능인 'Claude Code Routines'도 함께 선보였다. 한편, 최근 Claude의 토큰 소모가 빨라짐에 따라 스킬 사용량을 추적하고 관리할 수 있는 'skills-cleaner' 플러그인이 등장해 사용자들의 관심을 끌고 있다.

02

구글 Gemma 4 모델 공개 및 로컬 실행 사례 확산

구글의 최신 모델인 Gemma 4 시리즈가 허깅페이스를 통해 공개되었다. 31B-it, 26B-A4B-it, 그리고 any-to-any 기능을 갖춘 E4B-it 모델 등이 포함되었으며, 이미지-텍스트-텍스트 파이프라인을 지원한다. 특히 Gemma 4를 클라우드가 아닌 로컬 Codex CLI 환경에서 실행하여 도구 호출 성능과 안정성을 검증한 사례가 보고되었다. M4 Pro 칩을 탑재한 Mac이나 NVIDIA GB10 환경에서 실행한 결과, 기존 GPT-5.4 대비 비용 절감과 프라이버시 보호 측면에서 이점이 있는 것으로 확인되었다.

03

AI 에이전트 인프라 확장과 '바이브 코딩'의 보안 경고

AI 에이전트의 안전한 네트워크 접근을 위해 Cloudflare Mesh가 출시되었다. 이는 기존 VPN의 한계를 넘어 개인 디바이스와 원격 서버, 에이전트를 연결하는 경량 커넥터를 제공한다. 또한, LLM을 실제 에이전트로 만드는 오픈소스 인프라인 OpenHarness가 공개되어 도구 사용과 메모리 조율을 통합 제공한다. 하지만 AI 코딩 에이전트를 이용한 '바이브 코딩'의 위험성도 드러났다. 최근 한 의료기관 직원이 AI로 제작한 환자 관리 앱에서 데이터가 암호화 없이 노출되고 진료 녹음이 외부 서비스로 전송되는 보안 참사가 발생했다.

허깅페이스, Qwen3.5 및 차세대 멀티모달 모델 잇따라 등장

허깅페이스에 다양한 고성능 AI 모델들이 업데이트되었다. Qwen3.5-27B 모델을 기반으로 Claude 4.6 Opus의 추론 능력을 증류(Distilled)한 모델과 검열을 제거한(Uncensored) 버전들이 공개되어 주목받고 있다. 영상 생성 분야에서는 Lightricks의 LTX-2.3이 이미지-비디오 및 텍스트-비디오 변환 기능을 제공하며, 넷플릭스의 void-model은 비디오 인페인팅과 객체 제거 등 고도화된 비디오-비디오 편집 기능을 선보였다. 이 외에도 다국어 음성 합성 모델인 VoxCPM2 등이 추가되어 멀티모달 생태계가 확장되고 있다.

05

AI 감시 체계의 위험성과 안전성 평가 논의 가속화

AI를 이용한 대규모 감시 가능성에 대한 우려가 커지고 있다. 최근 유출된 검색 기록을 Claude로 분석한 결과, LLM의 뛰어난 독해 능력이 대중 감시의 강력한 도구가 될 수 있음이 시사되었다. 이와 관련해 AI 제어 평가(Control Evals)에서 공격자가 전략적으로 공격 방법을 선택할 경우 안전성이 저하될 수 있다는 연구 결과가 LessWrong을 통해 공유되었다. 또한, 앤스로픽의 Claude Mythos 프리뷰에 대한 분석에서는 사이버 보안 역량의 도약이 확인되었으며, 이에 따른 책임감 있는 확장 정책(RSP)의 변화가 논의되고 있다.

AMD ROCm, 엔비디아 CUDA 생태계 추격 가속화

AMD가 엔비디아의 CUDA 생태계에 대응하기 위해 AI 소프트웨어 스택인 ROCm을 중심으로 데이터센터 GPU 전략을 강화하고 있다. ROCm은 초기 단순한 펌웨어 묶음 수준에서 벗어나 이제는 완전한 소프트웨어 플랫폼으로 발전했으며, 6주 주기의 빠른 릴리스 체계를 도입해 업데이트 속도를 높이고 있다. 이는 AI 하드웨어 시장에서 엔비디아의 독점적 지위에 도전하고, 개발자들에게 더 넓은 소프트웨어 선택지를 제공하려는 전략의 일환으로 풀이된다.
